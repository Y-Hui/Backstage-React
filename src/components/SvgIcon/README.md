# SvgIcon

| Props     | 说明                                 | 类型   | 默认值 |
| --------- | ------------------------------------ | ------ | ------ |
| iconName  | Svg Symbol 文件名。例如：`#search`   | string | -      |
| color     | 设置 Svg 颜色。亦可使用 css 属性设置 | string | -      |
| className | 设置类名 | string | -      |
| style     | 设置行间样式 | 符合 React Style 对象类型 | -      |

| Event   | 说明     |
| ------- | -------- |
| onClick | 点击事件 |

备注：

建议使用 css 设置 svg 图标样式：

- color 设置颜色。

- font-size 设置图标大小。（图标默认 1em）
